
# Tic_Toc_Sujit_Game 🎮

A personalized version of the classic **Tic-Tac-Toe** game, built with Python and Flask.  
This project highlights clean UI design, modular code structure, and interactive gameplay.

---

## ✨ Features
- 🎲 Interactive 3x3 board with hover effects  
- 🔄 Player turn tracking (X and O)  
- 🏆 Automatic win detection and draw handling  
- ♻️ Reset button for replay  
- 📂 Organized project structure with `app.py`, `script.py`, and static assets
